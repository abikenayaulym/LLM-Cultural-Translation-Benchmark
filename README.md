# LLM-Cultural-Translation-Benchmark

A research project evaluating the performance of lightweight LLMs in retrieving culturally specific Chinese idioms (Chengyu) from Xianxia/Wuxia literature.

## Project Structure
- `dataset.csv`: The core dataset containing 18 domain-specific idioms and model responses.
- `analysis.py`: Python script used to process data and generate performance visualizations.

## How to run the analysis
1. Install dependencies: `pip install pandas matplotlib seaborn`
2. Run the script: `python analysis.py`
